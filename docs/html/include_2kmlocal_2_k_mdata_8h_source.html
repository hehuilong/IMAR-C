00027 //----------------------------------------------------------------------
00028 //  KMdata - data point set
00029 //      This object represents a set of data points in d-space.  The
00030 //      array can be resized.  Doing so destroys the existing contents.
00031 // 
00032 //      In addition to the points, we also (optionally) provide a
00033 //      kc-tree data structure for the points as well.  This is
00034 //      constructed by first initializing the points and then calling
00035 //      buildKcTree().
00036 //
00037 //      We support a virtual function samplePt and samplePts, which
00038 //      sample one or a set of random center points.  In this version,
00039 //      the sample is just a random sample of the point set.  However,
00040 //      it is possible to derive classes from this in which sampling is
00041 //      done by some more sophisticated method.
00042 //
00043 //      Note that this structure does not support copying or
00044 //      assignments.  If you want to resuse the structure, the only way
00045 //      to do so is to first apply resize(), which destroys the kc-tree
00046 //      (if it exists), and then assign to it a new set of points.
00047 //----------------------------------------------------------------------
00048
